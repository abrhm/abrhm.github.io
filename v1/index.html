The Code

v0

  1. Create a new folder and name it something e.g. meta-blog
  2. Now we would like to track changes and previous versions so use git with git init
  3. I chose github to host this fancy blog, you can do that by creating a repository with a name {github_user_name}.github.io so in my case abrhm.github.io
  4. I've created an empty index.html file and generated its content using VSCode autocomplete with a single "!" cool!
  5. Add the file to the version control and push and it is working! (sort of)
    So let's check what we generated with VSCode GitHub:
    <!DOCTYPE html>         // We would like to use HTML5
    <html lang="en">        // We are using english language
    <meta charset="UTF-8">  // We are using UTF-8 characters
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    Whoa whoa, this is my first stop this I bet half of the developers or even more don't understand what this means (including me)
    So let's break it up
    We are setting the viewport (how the website will look like)
    The content (which you are seeing) should be
    width=device-width  // As wide as possible by the device and
    initial-scale=1.0   // The initial scale is 1.0 but it can be further zoomed
    To be honest I've tried to play around with the initial-scale but there were no effect.
    Also there are some other interesting things like interactive-widget which you can play around with.
